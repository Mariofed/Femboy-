# 🎀 Femboy++ Ultimate (FBP)

Un linguaggio di programmazione **high-level**, estetico e interattivo, progettato per chi vuole programmare con stile. Basato su un'architettura viola e rosa, Femboy++ unisce la logica del C++ a una sintassi moderna e "slay".

## 🚀 Come iniziare
Per eseguire il tuo codice Femboy++, usa l'editor ufficiale `femboy_ultimate.html` sul tuo browser o lancialo tramite il compilatore integrato con:
`executeFemboy(); // Slay the code! ✨`

## 💎 Caratteristiche Principali
* **40+ Parole Chiave**: Un dizionario completo per logica, matematica e cicli.
* **Console Interattiva**: Gestione dell'input direttamente nel terminale con il comando `hear`.
* **Auto-Librerie**: Caricamento automatico di `aesthetics` e `official` packages.
* **Ahh Support**: Espressioni emotive integrate nel flusso di codice.

## 💻 Esempio di codice (femboy.fbp)

it_boy main() {
    thread utente;
    smol livello soft 99;
